# Statify

A Laravel based self-hosted websiste statistics app

## Features
[ ] Import data from Google Analytics
[ ] Multiple websites monitoring by teams
[ ] Not blocked by adblockers
[ ] Self hosted
[ ] CSV Exports
[ ] Monitor site down
[ ] Fast
